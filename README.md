# ADS GD Pin Automation

A personal automation tool for creating and publishing pins to a single Pinterest account.

## What this does

This tool takes a product name and an Amazon affiliate link as input, then:
- Generates a pin title and description
- Sources or generates a matching image
- Publishes the pin to Pinterest via the official Pinterest API (v5)

## Status

Personal project — used for a single, personally-owned Pinterest account. Not a public or multi-user application.

## Tech

- Python
- Pinterest API v5 (OAuth 2.0)
